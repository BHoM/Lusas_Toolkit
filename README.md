# Lusas Toolkit 
## System Requirements
- Lusas Modeller v17.0
- Microsoft Visual Studio Community 2015 or higher
- GitHub for Desktop (Windows)
- Microsoft .NET Framework 4.6 and above (included with Visual Studio Community 2015)
- BHoM Release 2.0
